		- 계산중 사용자 정지 발생시.
		- 모달 없이 프로세스 처리중 정지 발생시.
	
	: 측정 클래스 설계
		- 각  형상별 측정 파라메터 및 클래스.
		- 전역(공용) 측정 파라메터 및 클래스.
	
	: theCommand 분류 처리
		- 분리 및 구조화, Action Command.
	
	: 라이브러리 정리
		- NM 에서 사용하는 유틸리티 라이브러리 정리.
		- NV 에서 사용하는 유틸리티 라이브러리 정리.
		- NM 에서 사용하는 DFC 정리.
		- NV 에서 사용하는 DFC 정리.

	: 장비상태 정보 자료구조.

	: 통신상태 정보 자료구조.

	: Option 관리 클래스 설계 (m_hasXXX 자료 처리).

		- 좌표계 업데이트, 재측정 결과 업데이트 처리 문제.
		- 형상요소 그래픽에 따른 과부하 문제.
		- 부모, 자식 관계 정보 보유.

		- 측정환경 정보 처리.(좌표계, 프로브, 측정속도, 이동속도, )

[프로브]
	: 프로브 자료구조 설계 (접촉_터치, 접촉_스케닝, 비접촉_터치, 비접촉_스캔, 레이저_포이트, 레이저_스캐너, 기타 센서 입력 데이터 처리.)

	: 프로브 교정경로 생성 : 상대위치를 이용한 교정 경로 추출 알고리즘.
	: 프로브 설치구조 설계 : 모듈 단위, CMM스캐일-TRF, 헤드설치-TRF, 모듈, 바디, 스타일러스, 스타, 크랭크 구조 설계. 
	: 교환장치 자료(설치)구조 설계 : 타입, 포트, 위치, 프로브컴포넌트 인덱싱 정보, 
	: 교환장치 컨포넌트 설치 및 세팅 인터페이스 (전체 재설계 및 개발) : (김상수) 분석 및 개발, 테스트 경험 있음. (UCC 컨포넌트 개념 배워야함.)
	: 교환장치 자동 초기화 구조 설계 : 타입별 경로 처리. 하드웨어 길이정보, 옵셋량에대한 구조 처리.
